AI Content Detective
Instructions to Install and Run:
Clone this Github repo
Ensure you have the latest version of Python installed
In your terminal:
    Install the dependencies: pip install flask flask-cors tensorflow tf-keras transformers opencv-python numpy keras
    Run the Flask server: cd backend
                          python server.py
    Ensure server.py was run from within the ./backend folder

In your browser: Open index.html from the ./frontend folder
After the Flask server is started, the project will work.
